> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 5 件局外组新物品(4 装备 + 1 消耗品)接进现有 `GT_MetaCatalog` + loadout 体系, 用"触发标签 + 现有触发点钩子派发"实现被动/主动效果, BasicDebug/163 零影响。

**Tech Stack:** UE 5.7 纯 C++; 验证 = `Build.bat`(exit 0) + `GT_RuntimeSmokeRunner` 163 冒烟 + 无头 `gt.*` 命令(`gt.Bag` 读 Hp/Power, `gt.Snapshot`/`gt.Status` 读战斗/协议)。本仓库**无单元测试框架**, 每个任务的"测试" = 编译 + 163 + 针对性无头验证。

**分支:** `docs/s6-items-design`(已含 S3/S4/S5: GT_MetaCatalog + 部署终端 + loadout)。

bool UGT_RunContext::TryGrantChestMagnetLoot(int32 X, int32 Y)

	if (!bLoadoutChestBonusLoot || !IsRunActive()) { return false; }
	const FIntPoint Coord(X, Y);
	if (ChestBonusGrantedCells.Contains(Coord)) { return false; }
	ChestBonusGrantedCells.Add(Coord);

	// 确定性选一件低价值回收物(用现有 LootRules/ItemCatalog 的低档表; 先 Read GT_LootRules.h 找现成 helper)。

	if (ItemId.IsNone()) { return false; }
	RunInventory.AddCarriedItem(ItemId, 1, FName(TEXT("recovered")));
	return true;
}
